🚀 Overview
This project uses Selenium to automate a browser, scrape video URLs from a Dailymotion channel, and analyze the frequency of characters in video IDs.

⚙️ Features
Automated browser interaction
Infinite scroll handling
Extracts up to 500 video URLs
Parses video IDs from URLs
Performs character frequency analysis
Outputs most common character

🛠️ Tech Stack
Python
Selenium
Collections (Counter)
Chrome WebDriver

▶️ How It Works
Opens Dailymotion channel page
Scrolls to load videos
Extracts video URLs
Retrieves video IDs
Combines all IDs
Counts character frequency
Outputs most frequent character
